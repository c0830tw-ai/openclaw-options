1. 系統概觀與資料流

OpenClaw Options 是一個自動化的選擇權避險決策輔助工具，每 5 分鐘從永豐 Shioaji 抓取台指選擇權鏈、TX 期貨、加權指數、台積電 / 0050 現股報價，計算多種領式（collar）結構，並透過本地網頁 + Firebase 即時顯示。

核心資料流

[永豐 Shioaji API]
       ↓ (snapshot + kbars)
[shioaji_collar.py]  ← 計算 ATR/BB/HV/ADX/IV/delta，挑選最佳履約價
       ↓
[latest_collar.json] (本機快取)
       ↓                    ↓
[server.py :8081]    [Firebase RTDB]
       ↓
[web/index.html]  ← React + Babel CDN，無需 build

★ Put 進場參考線（藍框）

這是一條水平視覺化條，顯示 TX 現價與 5 條歷史關鍵價位的距離百分比。

讀法：
  • 60M 下軌（綠）：60 分鐘 K 棒 BB(20) 下軌，超賣訊號
  • 30M 下軌（淺綠）：30 分鐘 K 棒 BB(20) 下軌
  • 10 日線（灰）：10 日均價，中期支撐
  • 30M 中軌（黃）：30M MA20
  • 60M 中軌（橘）：60M MA20
  • 現價（亮黃，大圓點）：TX 即時
買 Put 訊號（你的習慣）：
  1. 現價跌到 30M 下軌 → 短期超賣，可能是反彈點，但也是「跌勢確認」進場 put 的時機
  2. 現價跌到 60M 下軌 → 中期跌破，put 多買一點
  3. 現價跌破 10 日線 → 中期轉弱，hedge 加碼
  4. 現價在 30M / 60M 中軌之上 → 短期偏多，按兵不動

3.2 範圍溫度計（2330 / 0050）

每個標的下方各有一條溫度計。所有 marker 用「離現價 %」當共同 X 軸，所以可同時看到不同 scale 的 BB 通道（2330 點位）跟選擇權履約（TX 點位）。

2330 範圍溫度計示意圖（淺綠 = 向下保護點、亮黃 = 上限封頂點）

判讀邏輯

  • 現價靠近 BB↑（紅）：超買，注意拉回，賣 call 收 premium 機會佳
  • 現價靠近 BB↓（綠）：超賣，反彈機會，但跌破代表結構轉弱
  • Put 履約離現價愈遠：保護愈淺、premium 愈低
  • Call 履約離現價愈近：上限愈低、premium 愈高（收得多但容易被軋）
注意：2330 / 0050 的 BB 通道是「個股」尺度，但 Put / Call 履約是「TAIEX」尺度。X 軸統一用「距現價 %」所以可疊在同一條。% 數值的「離現價」中，Put / Call 的基準是 TX 期貨價。

10. 常見情境決策流程

情境 A：TX 突然急跌 -3%

  1. 看 PutRefsBar：現價是否跌破 60M 下軌？
  2. 看 StrikeAlert：是否觸發 🔴？
  3. 看 IV percentile：> 70 已經貴了，等回穩；< 50 還能加 put
  4. 看 Payoff 圖：當前 protective_put 結構是否還有保護厚度？
  5. 動作：
    • 跌破 60M 下軌 + IV < 50 → 加買 1 口 put（30 DTE，Δ -0.10）
    • 跌破 60M 下軌 + IV > 70 → 不追，等反彈再進

情境 B：盤整一週，IV 偏低

  1. regime 顯示「😴 盤整」+ 主推 hui-full
  2. 看 0050 領口建議結構：選 hui-full（賣 1 call + 買 2 put）
  3. 用 SpreadCard 找最划算的 strike 組合
  4. 用 PayoffMini 確認到期損益曲線形狀
  5. 下單前用 SimTable 跑 5 個情境模擬

情境 C：DTE 剩 10 天，put delta 變很小

  1. RollPanel 應該已顯示 high 換倉警示
  2. 看遠月 (SectionFarMonth) 推薦履約是哪個
  3. 用 RollPanel 估算淨成本（平倉收 - 開倉付）
  4. 確認新倉 IV 不要太貴（< 70 percentile）
  5. 動作：賣掉舊月 put + 買進新月 put（同一口）

⚠ 風險提醒

  • TXO 領式對沖的是加權指數系統性風險，無法涵蓋台積電或 0050 的個股特定風險。當標的因個別消息劇烈波動而大盤未同步下跌時，Put 保護效果有限。
  • 夜盤流動性低，選擇權報價可能與真實成交差距大，建議重要決策在日盤完成。
  • 本系統僅輔助決策，最終下單請自行判斷並承擔風險。
  • 回測樣本只有 9 個月（4 季），統計顯著性不足，請當作參考而非保證。
